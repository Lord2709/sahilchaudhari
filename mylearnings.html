Encoding Techniques for Machine Learning

9 September 2024
Encoder Characteristics Best Suitable for ML Models How to Import Libraries
Label Encoder Encodes categories as unique integers. Not suitable for unordered categories. Tree-based models (e.g., Decision Trees, Random Forest) from sklearn.preprocessing import LabelEncoder
OneHot Encoder Creates binary columns for each category. Can lead to high dimensionality. Linear models (e.g., Logistic Regression), Neural Networks from sklearn.preprocessing import OneHotEncoder
Ordinal Encoder Encodes categories as integers based on a defined order. Suitable only for ordered categories. Tree-based models, Linear models (if categorical variables are ordinal) from sklearn.preprocessing import OrdinalEncoder
Target Encoder Encodes categories based on the mean of the target variable. May introduce data leakage if not used with care. Regression models, CatBoost, Gradient Boosting from category_encoders import TargetEncoder
Binary Encoder Converts categories to binary code. Suitable for high-cardinality features. Tree-based models, Linear models, Gradient Boosting from category_encoders import BinaryEncoder
Leave-One-Out Encoder A variant of Target Encoding that avoids data leakage by excluding current observation. Regression models, Gradient Boosting from category_encoders import LeaveOneOutEncoder
Hashing Encoder Uses a hash function to map categories to a fixed number of features. May lead to collisions. Neural Networks, Random Forest, High-cardinality cases from category_encoders import HashingEncoder
Count Encoder Encodes categories based on the frequency of occurrence. Simple and intuitive. Tree-based models, Gradient Boosting from category_encoders import CountEncoder
Frequency Encoder Similar to Count Encoding, but uses the normalized frequency (probability). Tree-based models, Gradient Boosting from category_encoders import FrequencyEncoder
CatBoost Encoder Advanced target encoding specifically designed for CatBoost. Avoids overfitting and leakage issues. CatBoost, Gradient Boosting from category_encoders import CatBoostEncoder

Key Highlights:

  • OneHot Encoder works best with models that handle high-dimensional data well (like linear models).
  • Target Encoder and CatBoost Encoder are more powerful for models that need to learn relationships between categories and the target variable (like gradient boosting models).
  • Binary Encoder and Hashing Encoder are great when dealing with high-cardinality features, especially in memory-constrained environments.
  • Label Encoder is mainly suited for tree-based models that can handle numerical inputs directly.
